5. markdown中图片太大影响体验怎么办？  
   一个简单的解决方案是：文件->偏好设置->外观->打开主题文件夹->找到你使用的主题对应的css文件（如github.css)，在末尾加上如下代码片段，并重启typora即可。
   ```css
   img {
       zoom: 50%;
   }
   ```
